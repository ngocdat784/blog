## Java là gì?

Java là một ngôn ngữ lập trình hướng đối tượng, được sử dụng rộng rãi trong
phát triển phần mềm, đặc biệt là các hệ thống lớn và lập trình mạng.

## Đặc điểm nổi bật của Java
- Chạy trên nhiều nền tảng (Write Once, Run Anywhere)
- Hướng đối tượng
- Bảo mật cao
- Cộng đồng lớn

## Ứng dụng của Java
- Ứng dụng web (Backend)
- Ứng dụng desktop
- Lập trình mạng
