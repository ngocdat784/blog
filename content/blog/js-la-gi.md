## JavaScript là gì?

JavaScript là ngôn ngữ lập trình dùng để tạo các trang web động,
giúp website tương tác với người dùng.

## JavaScript chạy ở đâu?
- Trình duyệt web
- Server (Node.js)

## Ứng dụng của JavaScript
- Xử lý sự kiện
- Kiểm tra form
- Xây dựng ứng dụng web
