Lập trình hướng đối tượng trong Java

Dec 22, 2025

Lập trình hướng đối tượng là gì?

Lập trình hướng đối tượng (OOP) là phương pháp lập trình dựa trên các đối tượng và lớp.

4 đặc tính chính của OOP trong Java

  • Đóng gói (Encapsulation)
  • Kế thừa (Inheritance)
  • Đa hình (Polymorphism)
  • Trừu tượng (Abstraction)

Ví dụ đơn giản

class Student {
    String name;
    int age;
}
